Brakeman 0.9.1 Released

Nov 18th, 2011

A new vulnerability was disclosed yesterday in the Rails translate helper.

This vulnerability affects Rails 2.3.x when using the rails_xss plugin, Rails 3.0 - 3.0.10, and Rails 3.1 - 3.1.1.

The only change in the 0.9.1 release is a check for this new vulnerability.

Posted by David A. Wheeler (with previous work by Justin Collins) Nov 18th, 2011

Comments

Brakeman is an open source vulnerability scanner specifically designed for Ruby on Rails applications. It statically analyzes Rails application code to find security issues at any stage of development.

Brakeman Pro is the commercial version of Brakeman which offers a GUI, test integration, deeper analysis, and more.
