Welcome to Railroader!

Welcome to Railroader! Railroader is a security static analysis tool for analyzing applications that use Ruby on Rails. It is open source software (OSS) using the MIT license. It’s easy to install and use. Railroader is a fork of the Brakeman tool, which was once OSS but is no longer. If you want Brakeman, go to the Brakeman website instead.
