Railroader

A security static analysis tool for Ruby on Rails applications

Welcome to Railroader! Railroader is a security static analysis tool for funding vulnerabilities in applications that use Ruby on Rails. It’s easy to install and use. It is open source software (OSS) using the MIT license; we love bug reports and contributions! You can learn more about Railroader.
